📊 Xero Bills — Contractor

Bill 1. One bill per contractor engagement, paid net (invoice total minus super). The super amount is noted on the first line; there is no separate super line for non-GST contractors. Import via Accounting → Purchases → Import and pay in the normal ABA batch.

🧾 Xero Bills — Super

Bill 2. One bill per super-eligible invoice, billed to the clearing-house supplier, single description line, coded to 478-C (BAS Excluded). Import alongside the contractor bills but pay as a separate ABA batch.

🏦 AustralianSuper SAFF (SuperStream)

The SuperStream Alternative File Format file you upload to AustralianSuper's Employer Portal / QuickSuper (Contribution Files → Upload File). Built to the QuickSuper Contribution CSV spec (columns A–AM); one row per super-eligible performer.
